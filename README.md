# React

#  Tarea 9: Feria de Software en línea

## Sobre el Proyecto

Este es el Producto Mínimo Viable desarrollado en React para la Tarea 9 de Diseño de Interfaces Usuarias.

## ⚙️ Ejecución del Proyecto

Sigue estos tres pasos para poner en marcha la aplicación:

1.  **Instalar dependencias:**
    ```bash
    npm install
    ```
2.  **Iniciar la aplicación:**
    ```bash
    npm start
    ```
3.  **Visualizar:** Abrir el navegador en **http://localhost:3000**.
